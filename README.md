## Overview

A mobile-responsive To-Do List application where users can create, edit, delete, search, and organize tasks by week. Each week displays a summary of open and completed tasks, helping users track their productivity.

## Features

* Create tasks with title, description, date & time, and optional priority
* Edit and delete tasks
* Mark tasks as completed or in progress
* Weekly grouping of tasks (Monday to Sunday)
* Weekly summary cards showing completed vs. open tasks
* Search tasks using keywords

## Tech Stack

* React (Vite)
* Tailwind CSS
* Local storage / custom API integration (if applicable)
* Deployment: Vercel

## Installation

```bash
# Clone the repo
$ git clone https://github.com/ayushdeep20/todo-ios

# Navigate to project directory
$ cd todo-ios

# Install dependencies
$ npm install

# Start development server
$ npm run dev
```
